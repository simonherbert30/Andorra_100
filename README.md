## 🗓️ Training Block Breakdown

### Phase 1: Base Building (Weeks 1-4)
- Building consistency and aerobic base
- 30-42km per week
- Focus: Steady runs and introducing trail elevation

### Phase 2: Volume Building (Weeks 5-8)
- Increasing weekly distance and elevation
- 50-65km per week
- Peak volume weeks 7-8 (60-65km)
- Focus: Long trail runs with significant elevation gain

### Phase 3: Maintenance (Weeks 9-10)
- Maintaining fitness while allowing recovery
- 55-60km per week
- Focus: Intensity and elevation practice

### Phase 4: Taper (Weeks 11-13)
- Progressive reduction in volume
- 50km → 40km → 25km
- Focus: Quality over quantity, recovery prioritized

### Phase 5: Race Week (Week 14)
- Very light shakeout runs
- Rest days built in
- **June 12**: RACE DAY - 80KM Trail Run with 3900m elevation

## 💡 Training Tips

### Pace Guidelines
- **Easy Runs**: 6:30-7:30/km (conversational pace)
- **Tempo Runs**: 6:00-6:30/km (comfortably hard)
- **Long Trail Runs**: 7:00-7:30/km (sustainable trail pace)
- **Recovery Run**: 7:30-8:30/km (very easy)
- **Race Target**: 10:00-11:00/km (conserve energy for elevation)

### Weekly Structure
**Monday-Thursday**: Road/urban running
- Easy runs on city streets
- Tempo or threshold work for fitness
- Short, focused sessions

**Friday-Sunday**: Trail/elevation focus
- One long run with climbing
- One trail recovery run
- Mix of continuous climbing and technical terrain

### Important Considerations
- Always warm up 10-15 minutes before runs
- Cool down 5-10 minutes after runs
- Cross-training recommended on rest days (yoga, strength, cycling)
- Nutrition: Practice fueling strategy on long runs
- Hydration: Trail runs 8+ km should include hydration pack

## 📊 Monitoring Your Progress

### Key Metrics to Track
- **Weekly mileage**: Use dashboard stats
- **Elevation climb**: Accumulating climbs similar to race
- **Pace consistency**: Maintain paces in specified ranges
- **Recovery**: How you feel after runs
- **Sleep & nutrition**: Critical for injury prevention

### Injury Prevention
- Never jump more than 10% increase in weekly volume
- Include 1-2 cross-training sessions per week
- Focus on core strength and stability work
- Address any pain immediately (rest 2-3 days if needed)
- Proper footwear critical for trail running
